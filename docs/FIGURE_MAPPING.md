# Figure Mapping & Compilation Guide

## How to compile the report

### Option A — Overleaf (recommended, zero install)
1. Go to https://overleaf.com → New Project → Upload Project
2. Upload `report.tex`
3. Create a folder named `figures/` in the project
4. Upload the 20 renamed images (see table below) into `figures/`
5. Click "Recompile" → done

### Option B — Local
```
pdflatex report.tex
pdflatex report.tex   # second pass for TOC and cross-references
```

> The document compiles even WITHOUT the images: missing figures render as a
> labelled box. This lets you check the text first, then drop the images in.
